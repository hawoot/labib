# planning/

My personal planning & idea space — **not code**. Brainstorms, decisions, and a
queue of features to build later, so good ideas from conversations don't get lost.

## feature-queue/
One Markdown file per future feature/idea. Pending ideas live here; when a
feature is built and shipped, its file moves to **`feature-queue/done/`**.

```
planning/
  feature-queue/        # ideas waiting to be built (one .md each)
    done/               # shipped — moved here when complete
```

Convention for a feature file: a short **Status** line at the top, the
**problem**, the **idea/knobs**, and any **open questions**. Keep it light.
